# CEDT Workspace Monorepo

This repository is an organized snapshot of the local `cedt` workspace. The root has been cleaned up so projects are easier to browse without changing their internal code layout.

## Top-level layout

- `academic/` — coursework, exercises, and assignment repositories
- `apps/` — app and product projects
- `experiments/` — scratch files and AI / competition experiments
- `tooling/` — developer tooling, configs, plugins, and utility repos
- `assets/` — top-level archives and binary assets

## Notes

- Large datasets, generated artifacts, and local runtime state are intentionally excluded from Git.
- See `EXCLUDED_LARGE_FILES.txt` for the main omitted paths.
- The previous remote state was preserved on the backup branch `backup/pre-sync-20260407-134312`.
